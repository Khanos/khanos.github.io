# Khanos.github.io

Personal bio website for Epilef Rodriguez (Khanos), built as a fast static site for GitHub Pages.

## Stack

- HTML5 (semantic structure)
- Modern CSS (mobile-first, flex/grid, CSS variables)
- No runtime framework

## Local preview

Open [index.html](index.html) directly in your browser, or run a quick static server:

```bash
python3 -m http.server 8080
```

Then visit `http://localhost:8080`.

## Editing guide

- Main markup: [index.html](index.html)
- Styles: [assets/css/style.css](assets/css/style.css)

## Deployment

This repo is served by GitHub Pages from the default branch root. Any push to `master` updates the live site.
